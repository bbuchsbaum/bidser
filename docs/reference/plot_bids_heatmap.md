# Plot BIDS data as a heatmap

Plot BIDS data as a heatmap

## Usage

``` r
plot_bids_heatmap(data, color_scheme = "viridis", highlight_missing = TRUE)
```

## Arguments

- data:

  Preprocessed BIDS data from prepare_bids_data_for_plot

- color_scheme:

  Color scheme to use

- highlight_missing:

  Whether to highlight missing data

## Value

A ggplot object
