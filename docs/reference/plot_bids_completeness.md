# Plot BIDS data completeness

Creates a heatmap showing data completeness across subjects and tasks.

## Usage

``` r
plot_bids_completeness(data, color_scheme = "viridis")
```

## Arguments

- data:

  Preprocessed BIDS data from prepare_bids_data_for_plot

- color_scheme:

  Color scheme to use

## Value

A ggplot object
