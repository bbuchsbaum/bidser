# Clear Example BIDS Dataset Cache

Clears the session-level cache of downloaded example BIDS datasets. This
can be useful to free up memory or force re-download of datasets.

## Usage

``` r
clear_example_bids_cache()
```

## Value

Invisible NULL

## Examples

``` r
# Clear the cache
clear_example_bids_cache()
#> Example BIDS dataset cache cleared
```
